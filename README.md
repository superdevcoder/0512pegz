# Instructions

## Clone

```bash
git clone https://github.com/GalloDaSballo/nextjs-magic-eslint-jest
```

## Install

```bash
yarn
```


## Start

```bash
yarn dev
```

## Architecture

See `context/UserContext.tsx` for implementation of Login, Logout and Session Persistence

See `components/Signup` for Login / Signup component
